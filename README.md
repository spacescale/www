## www

Public website and engineering blog for SpaceScale.

## Requires

- Astro v6
- TypeScript
- Tailwind CSS
- MDX
- SolidJS islands
- Bun

## Install dependencies

```sh
bun install
```

## Start the dev environment

```sh
bun dev
```

## Build

```sh
bun run build
```

## Preview production build

```sh
bun run preview
```
